# AstNet Update Service

AstNet-upd — специализированный сервисный модуль, предназначенный для обеспечения бесперебойного и автоматического обновления компонентов экосистемы AstNet на ОС Windows.

## ⚙️ Установка и интеграция
1. **Предварительные условия**: Убедитесь, что у вас установлен основной клиент [AstNet for Windows](https://github.com/AstNet/AstNet-Windows).
2. **Загрузка**: Скачайте актуальную версию модуля из раздела [Releases](https://github.com/AstNet/AstNet-upd/releases).
3. **Настройка**: Разместите исполняемый файл в корневой директории основного приложения. Модуль автоматически интегрируется в систему для фоновой проверки обновлений.

## 💳 Лицензирование
Данное программное обеспечение является вспомогательным закрытым компонентом AstNet.
* **Доступ**: Модуль функционирует только при наличии активной лицензии на основной продукт.
* **Получение доступа**: [Официальный Telegram-бот @astnetx_bot](https://t.me/astnetx_bot)

## 🛡 Безопасность и поддержка
* **Целостность**: Все обновления проходят проверку контрольных сумм перед установкой.
* **Техподдержка**: При возникновении проблем с автоматическим обновлением обращайтесь в [Telegram-канал](https://t.me/astnetx?direct).
